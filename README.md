<h1 align="center">🏥 EDA Insurance Data Analysis</h1>

<h2>📌 Project Overview</h2>
<p>
This project focuses on <b>Exploratory Data Analysis (EDA)</b> of an insurance dataset using Python.
The objective is to analyze customer demographics, medical factors, and insurance charges to uncover
patterns, trends, and relationships that influence insurance costs. Through statistical analysis and
data visualization, the project provides valuable insights for insurance companies and stakeholders.
</p>

<h2>🎯 Problem Statement</h2>
<p>
Insurance companies need to understand the factors affecting medical insurance charges to make better
pricing and risk assessment decisions. Manual analysis of customer data is time-consuming and may fail
to reveal hidden patterns. This project uses Exploratory Data Analysis (EDA) techniques to identify
key factors such as age, BMI, smoking habits, and region that impact insurance expenses.
</p>

<h2>📂 Dataset Description</h2>

<ul>
<li>Age</li>
<li>Gender (Sex)</li>
<li>BMI (Body Mass Index)</li>
<li>Number of Children</li>
<li>Smoking Status</li>
<li>Region</li>
<li>Insurance Charges</li>
</ul>

<h2>📊 Analysis Performed</h2>

<ul>
<li>Data Cleaning and Preprocessing</li>
<li>Missing Value Analysis</li>
<li>Univariate Analysis</li>
<li>Bivariate Analysis</li>
<li>Correlation Analysis</li>
<li>Outlier Detection</li>
<li>Distribution Analysis</li>
<li>Data Visualization</li>
</ul>

<h2>📈 Key Insights</h2>

<ul>
<li>Smoking status has a significant impact on insurance charges.</li>
<li>Higher BMI values are associated with increased medical expenses.</li>
<li>Insurance charges generally increase with age.</li>
<li>Regional differences have a relatively smaller impact on charges.</li>
<li>Customer demographics play an important role in risk assessment.</li>
<li>Data visualization helps identify trends and outliers effectively.</li>
</ul>
